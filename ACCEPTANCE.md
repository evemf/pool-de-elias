# Criterios de aceptación

- [x] Registro asigna nivel *Novel* por defecto si no se modifica.
- [x] Elegibilidad valida nivel, sexo y edad mostrando mensajes claros al bloquear una inscripción.
- [x] Una competición permite crear entre 7 y 10 subcompeticiones con herencia de tabla de puntos.
- [x] Actualizar resultados recalcula puntos y ranking automáticamente.
- [x] Dashboard del jugador muestra inscripciones, resultados y puntos acumulados.
- [x] Tema preparado para i18n, accesibilidad básica sin avisos PHP ni consultas excesivas.
